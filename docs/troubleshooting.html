Troubleshooting

Solutions to common issues with MurmurTone.

Startup Issues

App doesn't start / No tray icon

MurmurTone may be running but the icon could be hidden in the system tray overflow.

Solution

  1. Click the ^ arrow in your system tray to see hidden icons
  2. Look for the MurmurTone icon (shield with waveform)
  3. If not found, try running MurmurTone from the Start Menu
  4. Check Task Manager for "murmurtone.exe" - if running, end the task and restart

SmartScreen warning during installation

"Windows protected your PC" message appears when running the installer.

Solution

This is normal for new applications that haven't been widely distributed yet:

  1. Click "More info"
  2. Click "Run anyway"
  3. The installer will proceed normally

We're working on code signing to eliminate this warning in future releases.

Error: "Model not found"

The speech recognition model hasn't been downloaded yet.

Solution

  1. Ensure you have an internet connection (for first-time download only)
  2. Try a smaller model first (Quick) which downloads faster
  3. Check your firewall isn't blocking the download
  4. The model is stored in %APPDATA%\MurmurTone\models

Transcription Issues

Transcription not working / No text appears

You press the hotkey, speak, release, but nothing happens.

Solution

  1. Check microphone: Ensure your microphone is connected and set as default in Windows Sound Settings
  2. Test audio: Right-click tray icon → Settings → Test Microphone
  3. Disable noise gate: Temporarily disable the noise gate to see if it's filtering your voice
  4. Check hotkey: Make sure no other application is using the same hotkey
  5. Restart app: Right-click tray icon → Quit, then relaunch

First transcription is slow

The first transcription takes 30-60 seconds.

Solution

This is normal behavior. The speech model needs to load into memory on first use. Subsequent transcriptions will be much faster (typically under 2 seconds).

To speed up model loading:

  • Use a smaller model (Quick or Standard)
  • Enable GPU acceleration if you have an NVIDIA card
  • Ensure MurmurTone starts with Windows so the model is preloaded

Poor transcription accuracy

Words are frequently misheard or garbled.

Solution

  1. Upgrade model: Try Recommended or Professional for better accuracy
  2. Check microphone: Use a quality microphone, avoid laptop built-in mics
  3. Reduce noise: Record in a quiet environment or enable noise gate
  4. Speak clearly: Enunciate words and speak at a moderate pace
  5. Add vocabulary: Add frequently used technical terms to Custom Vocabulary
  6. Check language: Ensure the correct language is selected in settings

GPU Issues

GPU not detected / GPU options disabled

GPU acceleration options are grayed out in settings.

Solution

  1. Check GPU: MurmurTone requires an NVIDIA GPU with CUDA support
  2. Update drivers: Install the latest NVIDIA drivers from nvidia.com/drivers
  3. CUDA version: Ensure CUDA 12.x compatible drivers are installed
  4. Restart: Restart MurmurTone after installing drivers
Note AMD GPUs are not currently supported. CPU mode works on any machine and is still fast for most use cases.

Audio Issues

Wrong microphone being used

MurmurTone is recording from the wrong device.

Solution

  1. Right-click tray icon → Settings
  2. Go to Audio Settings
  3. Select your preferred microphone from the "Input Device" dropdown
  4. Click "Test Microphone" to verify

Audio feedback sounds not playing

You don't hear beeps or chimes during transcription.

Solution

  1. Check that audio feedback is enabled in Settings → Audio
  2. Ensure your system volume is not muted
  3. Check that the correct audio output device is selected in Windows

License Issues

License activation failed

Unable to activate your license key.

Solution

  1. Check internet: License activation requires a one-time internet connection
  2. Verify key: Ensure you've entered the license key correctly (no extra spaces)
  3. Check email: The key was sent to your purchase email address
  4. Contact support: Email support@murmurtone.com with your order details

Log Files

If you need to report an issue, please include the log file:

%APPDATA%\MurmurTone\murmurtone.log

To open this location:

  1. Press Win + R to open Run dialog
  2. Type %APPDATA%\MurmurTone and press Enter
  3. Find murmurtone.log
