Privacy & Security

Is my voice data sent to the cloud?

No, never. MurmurTone processes all speech recognition 100% locally on your computer. Your voice recordings and transcriptions never leave your machine. This is a core architectural decision, not just a setting you can toggle.

Does MurmurTone collect any data about me?

MurmurTone does not collect usage data, analytics, or telemetry. The only data we receive is:

  • License activation: Your license key and a machine identifier (Pro users only)
  • Purchase: Your email address (sent through Gumroad)

Your voice data, transcriptions, and usage patterns are never collected.

Can MurmurTone work completely offline?

Yes! After the initial model download (~75MB-1.5GB depending on model), MurmurTone works completely offline. You only need internet for:

  • First-time model download
  • One-time license activation (Pro users)

After that, you can transcribe without any internet connection.

Is it safe to use for confidential documents?

Yes. Since all processing happens locally, MurmurTone is ideal for:

  • Legal documents and attorney-client communications
  • Medical notes (HIPAA considerations)
  • Business confidential information
  • Personal journaling and notes

Your data security depends on your computer's security, not our servers.

Getting Started

How do I install MurmurTone?

Installation is simple:

  • Download the installer from murmurtone.com/install
  • Run the installer (click "More info" then "Run anyway" if SmartScreen appears)
  • MurmurTone will appear in your system tray
  • The first transcription will download the speech model

See our Getting Started Guide for detailed steps.

How do I use MurmurTone?

It's designed to be simple:

  • Hold Ctrl+Shift+Space (or your custom hotkey)
  • Speak clearly
  • Release the key
  • Your transcribed text is automatically typed

Works in any application - email, documents, chat, code editors, anywhere you can type.

Why is the first transcription slow?

The first transcription takes 30-60 seconds because the speech model needs to load into memory. This is normal. After the initial load, subsequent transcriptions are fast (typically 1-3 seconds).

Tips to speed up model loading:

  • Use a smaller model (tiny.en or base.en)
  • Enable GPU acceleration if you have an NVIDIA card
  • Set MurmurTone to start with Windows
What's the system tray icon for?

MurmurTone runs in the background and appears as an icon in your system tray (bottom-right corner of Windows). Right-click it to:

  • Open Settings
  • View transcription history
  • Access voice commands reference
  • Quit the application

Features

What is AI text cleanup?

AI text cleanup uses a local AI model (via Ollama) to polish your transcriptions. It can:

  • Fix grammar and punctuation
  • Format text professionally
  • Remove filler words (um, uh, like)
  • Improve sentence structure

This feature requires Ollama installed separately, but runs 100% locally - no cloud AI involved.

Does MurmurTone support languages other than English?

Yes! MurmurTone supports 18+ languages including:

  • Spanish, French, German, Italian, Portuguese
  • Chinese, Japanese, Korean
  • Russian, Arabic, Hindi
  • And many more

Use multilingual models (tiny, base, small, medium without .en) for non-English languages.

What is Translation Mode?

Translation Mode lets you speak in one language and have your text output in English. Speak French, get English text. Speak Spanish, get English text. This is useful for:

  • Non-native English speakers drafting English documents
  • Multilingual teams communicating in English
  • Quick translations while working
Can I transcribe audio/video files?

Yes! MurmurTone can transcribe audio and video files (MP3, WAV, M4A, MP4, etc.). Right-click the system tray icon then select "Transcribe File" and select your media file. Transcription happens locally and is saved to a text file.

What voice commands are supported?

MurmurTone supports commands for punctuation, navigation, and editing:

  • Punctuation: "period", "comma", "question mark", "new line", "new paragraph"
  • Navigation: "go to start", "go to end"
  • Editing: "delete that", "delete last word", "undo that", "select all"
  • Case: "all caps", "all lowercase", "capitalize"

See the full list in our Voice Commands Reference.

Technical

What are the system requirements?
  • OS: Windows 10 or Windows 11 (64-bit)
  • RAM: 8GB minimum, 16GB recommended
  • Storage: 500MB for app + 75MB-1.5GB for models
  • CPU: Any modern processor (Intel/AMD)
  • GPU: NVIDIA GPU with CUDA 12.x (optional, for faster processing)
Does MurmurTone work on Mac or Linux?

Currently, MurmurTone is Windows only. Mac and Linux versions may be considered in the future based on demand.

Do I need a GPU?

No, a GPU is not required. MurmurTone works well on CPU. However, if you have an NVIDIA GPU with CUDA support, you can enable GPU acceleration for faster transcription (especially with larger models).

AMD GPUs are not currently supported for acceleration.

Which speech model should I use?

MurmurTone offers several Whisper models:

  • tiny.en (~75MB): Fastest, lowest accuracy, good for quick notes
  • base.en (~145MB): Good balance for most users (recommended)
  • small.en (~485MB): Better accuracy, slower
  • medium.en (~1.5GB): Best accuracy, requires more resources

Models ending in .en are English-only but more accurate. Use models without .en for other languages.

Where is my data stored?

All MurmurTone data is stored locally in %APPDATA%\MurmurTone:

  • Settings: Your preferences and configuration
  • Models: Downloaded speech recognition models
  • Vocabulary: Custom words you've added
  • Logs: Application logs for troubleshooting

To delete all data, simply remove this folder or uninstall the application.

Why does SmartScreen warn about the installer?

Windows SmartScreen may show a warning because MurmurTone is a new application that hasn't been widely distributed yet. This is normal for any new software.

To proceed: Click "More info" then "Run anyway"

We're working on code signing to eliminate this warning in future releases.
